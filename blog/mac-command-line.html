Part 1: Understanding the Mac File System

Key Concepts:

  • User Folders: Located in /Users/, each user has a home directory (e.g., /Users/yourname).
  • Applications: Most apps are stored in the /Applications folder.
  • System Directories: Folders such as /Library, /System, and /usr hold system and user resources.
  • Paths: Absolute paths start with / and uniquely define a file‚Äôs location.

Part 2: Navigating the File System with Terminal

Open the Terminal by navigating to /Applications/Utilities or searching for it via Spotlight.

1. See Your Current Location

Command:


pwd

When to use: The pwd command prints your current working directory.

2. List Files and Folders

Command:


ls

Use ls -l for a detailed listing or ls -a to show hidden files.

3. Change Directory

Command:


cd /Users/yourname/Documents  # Move to Documents folder
cd ..                         # Move up one level
cd /                         # Go to the root directory

When to use: Use the cd command to navigate between directories.

4. Create New Folders

Command:


mkdir MyFolder

When to use: mkdir creates a new directory.

5. Create Files

Command:


touch myfile.txt

When to use: touch creates an empty file or updates its timestamp.

6. Delete Files and Folders

Command:


rm myfile.txt         # Delete a file
rm -r MyFolder        # Remove a folder and its contents

When to use: Use rm to delete files and rm -r to delete directories recursively.

7. Copy and Move Files

Copy Files:


cp myfile.txt /Users/yourname/Backup/

Move Files:


mv myfile.txt /Users/yourname/Backup/

When to use: Use cp to copy files and mv to move or rename them.

Part 3: Intermediate Command Line Power Moves

Once you're comfortable with the basics, move on to more intermediate commands.

1. Open Applications

Command:


open -a "TextEdit"

When to use: Use open with the -a flag to launch applications (here, TextEdit).

2. Check Disk Space

Command:


df -h

When to use: df -h displays disk usage in a human-readable format.

3. Find Files Containing Specific Text

Command:


grep "error" log.txt

When to use: Use grep to search for text patterns within files.

4. Copy Multiple Files at Once

Command:


cp *.txt /Users/yourname/Backup/

When to use: Wildcards allow you to copy multiple files at once.

5. Clean Up Temporary Files

Command:


rm -rf /tmp/*

When to use: Use rm -rf cautiously to remove temporary files and directories.

Key File Locations & Tools Available in the CLI

Now that you've learned how to navigate and manage files in the Terminal, it's important to know where to find key system files and the tools available in macOS. Here's a breakdown:

Key File Locations

Common directories in macOS include:

  • / - The root directory of the system.
  • /Applications - Where most applications are installed.
  • /Users/yourname - Your home directory containing Documents, Downloads, Desktop, etc.
  • /Library - Contains system and user libraries, preferences, and support files.
  • /System - Holds essential system files (modify with caution).

Note: Use commands like cd /Users/yourname to navigate quickly.

Essential Command Line Tools

Some fundamental commands for macOS Terminal include:

  • pwd - Prints the current working directory.
  • ls - Lists directory contents (ls -la shows hidden files and details).
  • cd - Changes the current directory.
  • mkdir - Creates new directories.
  • touch - Creates an empty file or updates its timestamp.
  • rm - Removes files (rm -r for directories).
  • cp - Copies files or directories.
  • mv - Moves or renames files or directories.
  • grep - Searches for text patterns within files.
  • open - Opens files or applications (e.g., open -a "Safari").

Note: Consult each command‚Äôs manual (e.g., man ls) for more details.

Environment Variables

macOS uses environment variables to simplify navigation. For example:

  • $HOME - Your home directory (/Users/yourname).
  • $PATH - A colon-separated list of directories where executables are located.

Use these variables (e.g., cd $HOME) to save time.
